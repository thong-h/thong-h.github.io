The Lantern & Horror Park (2020)

DESCRIPTION:

This project is to create a soundtrack, then make two contrast videos for that soundtrack. As Halloween is coming, I want to create a work around that main theme for one of my videos, and for the other, I aim for something bright that shows the opposite. The audio is based on the Halloween theme so it has a lot echos and delays in some specific sounds like laughs, howls, hums, and so forth in order to create a horrific atmosphere. Through this project, viewer can clearly see how sound can strongly impact on the visual images.

TOOL: Audition, Premiere Pro, Iphone [XR]

QUALITY: FHD 1080
